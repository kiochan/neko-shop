# Neko Shop

A simple e-commerce framework built with [Next.js](https://nextjs.org/).

## Run locally

```bash
# Clone the project
git clone https://github.com/kiochan/neko-shop.git

cd neko-shop

# Install dependencies
npm install

# Start the dev server
npm run dev
```
