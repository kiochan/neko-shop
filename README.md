# Neko Shop

A simple e-commerce framework built with Next.js.

## Requirements

- Node.js >= 20
- pnpm >= 10

## Run locally

```bash
git clone https://github.com/kiochan/neko-shop.git
cd neko-shop

# Install dependencies
pnpm install

# Start development server
pnpm dev
```
